What is FileSharer ?

FileSharer is a web application created for sharing files, and messages across the web.

How it works ?

FileSharer uses a NoSql DBMS that is mongoDB to store all the files and messages, whenever a user logs in all the files related to that user are fetched from the database and rendered on the screen.
